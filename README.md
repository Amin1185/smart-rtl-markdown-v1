# smart-rtl-markdown

Smart RTL-aware Markdown previewer — Version 1.0.0

A lightweight client-side Markdown previewer optimized for mixed RTL/LTR content, MathJax math rendering, and code syntax highlighting. This repository is an initial, standalone release (v1.0.0).

## Files
- `index.html` — main single-file web app previewer.

## Quick start
1. Open `index.html` in your browser.
2. Or serve locally (recommended) and visit `http://localhost:8080`:

```bash
npx serve .
```
